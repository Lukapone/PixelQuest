# PixelQuest

A Zig-based game learning project built with the Zig programming language.

## About

PixelQuest is a learning project for developing games using Zig. This project serves as an educational foundation for understanding game development concepts, Zig programming patterns, and modern systems programming.

## Getting Started

### Prerequisites

- Zig 0.15.2 or later
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Lukapone/PixelQuest.git
   cd PixelQuest
   ```

2. Build and run the game:
   ```bash
   zig build run
   ```

3. Run tests:
   ```bash
   zig build test
   ```
